# Vedrix AI Interview System

**Vedrix** is a modern AI-powered interview platform designed to revolutionize the recruitment process and candidate preparation.

## 🚀 Overview
Vedrix provides a dual-sided experience:
- **For Candidates (B2C):** Practice adaptive AI interviews with real-time feedback.
- **For Recruiters (B2B):** Create job drives and use AI as an unbiased screening layer.

## 🛠️ Technology Stack
- **Backend:** FastAPI (Python)
- **Frontend:** React (Vite, Tailwind CSS)
- **Database:** PostgreSQL (SQLAlchemy)
- **AI Integration:** OpenRouter (Groq, NVIDIA, DeepSeek)

## 📁 Project Structure
- `backend/`: FastAPI application.
- `frontend/`: React application.
